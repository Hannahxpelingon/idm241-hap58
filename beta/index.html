Beta Description

IKEA Card Carousel

IKEA Website

Triggers

Carousel Card

  • Hover On
  • Hover Off

Left Chevron Button

  • Hover On
  • Hover Off
  • On Click

Right Chevron Button

  • Hover On
  • Hover Off
  • On Click

Pagination

  • Hover On
  • Hover Off
  • On Click

Rules

Carousel Card

  • Hover On
    • Card background changes color to a blue (#1459A2)
    • Text moves slightly to the right
    • Arrow slides in from the left
  • Hover Off
    • Card background reverts back to white
    • Text moves slightly to the left, going back to original position
    • Arrow slides back to the left and hides

Left Chevron Button

  • Hover On
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cursor changes to finger
  • Hover Off
    • Button color reverts from a dark gray (#333333) back to black (#121212)
    • Cursor reverts to pointer
  • On Click
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cards on the carousel shift one to the left
    • Pagination shifts one to the left, unless the starting point is the first card
    • If the starting point is the second card and the user clicks the left chevron button, the background color changes to a light gray with a dark gray arrow.

Right Chevron Button

  • Hover On
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cursor changes to finger
  • Hover Off
    • Button color reverts from a dark gray (#333333) back to black (#121212)
    • Cursor reverts to pointer
  • On Click
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cards on the carousel shift one to the right
    • Pagination shifts one to the right, unless the starting point is the last card
    • If the starting point is the third card and the user clicks the right button, the background color changes to a light gray with a dark gray arrow.

Pagination

  • Hover On White Dots
    • Cursor changes to finger
  • Hover Off White Dots
    • Cursor reverts to pointer
  • On Click White Dots to the Right
    • Card carousel moves to the respective set of cards to the right
  • On Clicking the Last White Dot
    • Card carousel shifts the last set of cards
    • The background color of the right chevron button changes to a light gray with a dark gray arrow.
  • On Clicking White Dots to the Left
    • Card carousel shifts to the respective set of cards to the left
  • On Clicking the First White Dot
    • Card carousel shifts the first set of cards
    • The background color of the right chevron button changes to a light gray with a dark gray arrow.
