Rules

Slider Card

  • Hover On
    • Cursor changes to finger
  • Hover Off
    • Cursor reverts to pointer
  • On Click
    • Opens new page

Left Chevron Button

  • Hover On
    • If on first card, pointer changes to arrow with a no symbol on the bottom. The starting color is a background with light gray with a dark gray arrow.
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cursor changes to finger
  • Hover Off
    • Button color reverts from a dark gray (#333333) back to black (#121212)
    • Cursor changes to finger
  • On Click
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cards on the carousel shift to the left

Right Chevron Button

  • Hover On
    • If on last card, pointer changes to arrow with a no symbol on the bottom. The starting color is a background with light gray with a dark gray arrow.
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cursor changes to finger
  • Hover Off
    • Button color reverts from a dark gray (#333333) back to black (#121212)
    • Cursor changes to finger
  • On Click
    • Button color changes from black (#121212) to a dark gray (#333333)
    • Cards on the carousel shift to the right

Carousel Scroll Bar

  • On drag to the left
    • Cards on the carousel move left
    • If scrollbar indicator touches the left start: the right chevron button changes to a light gray background with a dark gray arrow
  • On drag to the right
    • Cards on the carousel move right
    • If scrollbar indicator touches the right end: the right chevron button changes to a light gray background with a dark gray arrow

Feedback

Intuit Mailchimp logo

  • Hover On
    • Light gray box surrounding the logo appears
    • Logo bounces up and down once
    • As the logo bounces, the monkey opens its eyes then returns to the winking (original position)
    • Cursor changes to finger
  • Hover Off
    • Cursor reverts to pointer
  • On Click
    • A small Intuit tab flashes from the top then disappears once the page is reloaded
